# 活动方案报价单生成器

这是一个纯本地运行的「方案内容转报价单」初版工具。

第一版不接入任何 AI API，不需要登录、不需要后端、不需要云服务。程序通过本地规则识别方案文本中的活动内容、物料、人员、搭建、演艺、设备等报价线索，统一成标准报价项，再由人工编辑并导出 Excel 报价单。

## 功能范围

- 支持粘贴方案文本。
- 支持上传 `.txt` / `.docx` / `.pdf` 文件并读取文本。
- 按 `data/rules_config.json` 中的同义词规则识别报价项。
- 将“市集摊位”等统一为“帐篷摊位”。
- 将“开场表演、乐队演绎、舞蹈演绎、脱口秀演绎”等统一为“节目演出”。
- 读取空报价库模板 `data/price_db.xlsx`，后续可补充默认规格、单位、单价。
- 使用 `st.data_editor` 人工编辑报价明细。
- 使用 `openpyxl` 导出正式版式的 Excel 报价单。

## 本地运行

```bash
cd quote_app
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

```

## 报价库说明

`data/price_db.xlsx` 是空模板，不包含真实价格。

字段如下：

```text
项目分类
标准项目
默认规格
单位
默认单价
报价类型
备注
```

如果价格库中能匹配到“项目分类 + 标准项目”，程序会带出默认规格、单位、默认单价、报价类型和备注。第一版默认单价可以为空；界面中识别项会先显示为 0，方便人工补充。

## 测试文本

活动名称：农野开心乐  
活动时间：2026年6月  
活动地点：海口自在湾

活动现场设置区域A为农产品市集区，计划设置10个市集摊位，每个帐篷约2m×2m，搭配桌椅、摊位牌、木箱、竹筐、麻布、稻穗及暖色串灯进行氛围布置。

区域B为舞台区，舞台长8m、纵深3m，背景板高度2.4m，采用异形PVC板。舞台配置线阵音响，灯光配置包含8个帕灯、2个射灯、5盏面光灯。

活动期间安排开场表演、乐队演绎、舞蹈演绎及脱口秀演绎。现场设置工作人员、安保人员、工作证、对讲机、饮用水、餐费及运输服务。

使用上面的测试文本，初版应能识别出：

- 帐篷摊位
- 氛围布置
- 舞台搭建
- 舞台背景
- 音响设备
- 灯光设备
- 节目演出
- 执行人员
- 安保人员
- 现场工作证
- 对讲机
- 饮用水
- 餐费
- 运输费

其中“市集摊位”会归一为“帐篷摊位”；“开场表演、乐队演绎、舞蹈演绎、脱口秀演绎”会归一为“节目演出”。
